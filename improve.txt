# 5️⃣ Add Data Encryption 🔐

## Current Problem

Data is visible in localStorage.

## Improvement

Encrypt before storing.

## Example

```js
btoa(JSON.stringify(data))
```

Decode:

```js
JSON.parse(atob(data))
```

## Advanced Future Upgrade

* AES encryption
* Password-protected vault

# ❌ Avoid innerHTML += Inside Loops

## Current

```js
this.list.innerHTML +=
```

## Better Way

```js
let html = "";

contacts.forEach(c => {
   html += `...`;
});

this.list.innerHTML = html;
```

## Why?

Faster rendering.
