# 📒 Contact Book -- Contact Management System

A modern and responsive **Contact Management Web Application** built
using **HTML, CSS, and JavaScript**.

This project allows users to add, update, delete, and search contacts
with a clean UI, theme modes, and local storage support.

------------------------------------------------------------------------

## 🚀 Features

-   Add new contacts\
-   Edit existing contacts\
-   Delete contacts\
-   Search by name or phone\
-   Live total contact counter\
-   Dark Mode toggle\
-   RGB Animated Theme\
-   Toast notifications\
-   Data stored using LocalStorage\
-   Responsive design

------------------------------------------------------------------------

## 🛠 Technologies Used

-   HTML5\
-   CSS3 (Flexbox, Grid, Animations)\
-   Vanilla JavaScript (ES6)\
-   LocalStorage API

------------------------------------------------------------------------

## 📌 How It Works

### 1️⃣ Storage Module

Handles saving and retrieving contacts from browser LocalStorage.

### 2️⃣ Validation Module

-   Validates Indian mobile numbers (Starts with 6--9)
-   Prevents duplicate phone numbers

### 3️⃣ UI Module

-   Renders contact list dynamically
-   Controls slide form
-   Shows toast notifications

### 4️⃣ App Module

-   Controls full application logic
-   Add / Edit / Delete operations
-   Theme persistence
-   Live stats update

------------------------------------------------------------------------

## 🔐 Validation Rules

-   Name is required
-   Phone must be 10 digits
-   Phone must start with 6--9
-   Duplicate contacts are not allowed

------------------------------------------------------------------------

## ▶️ How to Run the Project

1.  Download or clone the project
2.  Open `index.html`
3.  Click **Enter**
4.  Start adding contacts

No backend or server required.
